Skills

Adding a skill​

Create a new Blueprint class with a type of Tree Skill. Call this the name of your base skill (not perk)

Open up the newly created skill and set the Display Name and Description.

Add this new skill to your Skill Tree component's Skill Tree Skills.

You can now add perks to this skill via the Adding new perks.
