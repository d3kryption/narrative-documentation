# Skills

## Adding a skill

Create a new Blueprint class with a type of `Tree Skill`. Call this the name of your base skill (not perk)

![SkillTree_AddSkill_Class.webp](//img/pro/skill-trees/SkillTree_AddSkill_Class.webp)
![SkillTree_AddSkill_BP.webp](//img/pro/skill-trees/SkillTree_AddSkill_BP.webp)

Open up the newly created skill and set the Display Name and Description.

![SkillTree_AddSkill_Details.webp](//img/pro/skill-trees/SkillTree_AddSkill_Details.webp)

Add this new skill to your [Skill Tree component](./index.md)'s Skill Tree Skills.

![SkillTree_AddSkill_Skills.webp](//img/pro/skill-trees/SkillTree_AddSkill_Skills.webp)

You can now add perks to this skill via the [Adding new perks](./adding-new-perks.md).
